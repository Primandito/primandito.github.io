Auth Bypass & Critical IDOR (PII Data Leakage)

Aplikasi VIERA - PT International Test Center (ITC) Indonesia
Valid Critical

A critical security vulnerability affecting over 100,000 student records has been identified. This vulnerability allows for full authentication bypass and Insecure Direct Object Reference (IDOR), allowing unauthorized access and manipulation of sensitive PII data (NISN, email, phone number, and other sensitive data). It has been addressed and acknowledged by the ITC technical team.
